# 🎨 Random Color Generator

A simple JavaScript project that changes the background color randomly every second.

## 🚀 Features

- Generates random HEX colors
- Background changes every 1 second
- Start and Stop buttons
- Built using HTML, CSS, and JavaScript

## 📂 Project Structure

```
random-color-generator/
│
├── index.html
├── index.css
├── index.js
└── README.md
```

## 🛠 How It Works

- A function generates a random HEX color
- `setInterval()` changes the background every second
- `clearInterval()` stops the color changing

## 🧠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🎯 Learning Purpose

This project was created to practice:
- DOM manipulation
- setInterval and clearInterval
- Event listeners
- Random color generation
